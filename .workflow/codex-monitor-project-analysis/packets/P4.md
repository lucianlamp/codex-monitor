# P4 Verification

Objective: Run local non-destructive verification commands and report current
health.

Commands:

- `cargo fmt --check`
- `cargo check`
- `cargo test`
- `cargo clippy --all-targets -- -D warnings`
- `cargo check --target x86_64-pc-windows-msvc`
- `cargo check --tests --target x86_64-pc-windows-msvc`
- `cargo run --bin cdxm -- threads --cwd /Users/ysk411/dev/codex-monitor`

Do not: run live mutating `send` or start agmsg watchers.

Expected output: pass/fail status and exact skipped checks.

Verification: workflow verifier and git status.
